# 能源与电力系统知识库

## 1. 智能电网技术

### 1.1 智能电网概述

智能电网（Smart Grid）是将先进的传感技术、通信技术、信息技术和控制技术与传统电力系统深度融合的新型电网形态。其核心目标是实现电网的可靠、安全、经济、高效运行，同时支持大规模可再生能源接入。

智能电网的主要特征包括：
- **自愈能力**：能够自动检测、分析并响应电网故障，快速恢复供电
- **用户互动**：支持用户参与电网运行，实现需求侧响应
- **分布式能源接入**：兼容光伏、风电、储能等多种分布式能源
- **高电能质量**：提供满足数字时代要求的电能质量
- **优化资产利用**：通过数字化手段提升电网资产运营效率

### 1.2 高级量测体系（AMI）

高级量测体系（Advanced Metering Infrastructure, AMI）是智能电网的关键基础设施，由智能电表、通信网络和数据管理系统组成。

| 组件 | 功能 | 技术指标 |
|------|------|----------|
| 智能电表 | 双向电能计量、实时数据采集 | 精度等级 0.5S，采集间隔 ≤ 5 min |
| 通信网络 | 数据传输、远程控制 | 覆盖率 ≥ 99.5%，延迟 ≤ 3s |
| 数据管理系统 | 数据存储、分析、展示 | 处理能力 ≥ 100 万户/小时 |
| 家庭网关 | 用户侧数据汇聚 | 支持多种通信协议 |

AMI 系统的核心价值：
1. 实现精确的用电量计量和计费
2. 支持分时电价和实时电价的实施
3. 为需求侧响应提供数据基础
4. 提高电网运行的可视化水平

### 1.3 配电自动化

配电自动化（Distribution Automation, DA）利用自动化技术和通信技术，实现对配电网的监测、控制和保护。主要功能包括：

- **故障检测与隔离**：在毫秒级时间内检测故障并自动隔离故障区段
- **网络重构**：根据负荷变化动态调整配电网拓扑
- **电压无功控制**：自动调节电压和无功功率，提高供电质量
- **负荷均衡**：优化负荷分配，防止设备过载

## 2. 新能源发电技术

### 2.1 光伏发电系统

光伏发电系统将太阳能直接转化为电能，是目前增长最快的可再生能源技术之一。

#### 2.1.1 光伏电池类型

| 类型 | 转换效率 | 成本 | 适用场景 | 寿命 |
|------|----------|------|----------|------|
| 单晶硅 | 22-26% | 较高 | 大型地面电站、屋顶 | 25-30年 |
| 多晶硅 | 18-22% | 中等 | 地面电站 | 25年 |
| 薄膜电池 | 10-15% | 较低 | BIPV、柔性应用 | 20年 |
| 钙钛矿 | 25-28% | 低（潜力） | 实验室/中试 | 待验证 |

#### 2.1.2 光伏电站并网技术

光伏电站并网需要解决以下关键技术问题：

1. **功率波动性**：光伏出力受天气影响，波动范围可达额定容量的 0-100%
2. **电能质量**：逆变器产生的谐波需控制在 IEEE 519 标准限值内
3. **低电压穿越（LVRT）**：电网电压跌落时，光伏电站需维持并网运行
4. **频率支撑**：具备一定的调频能力，参与电网一次调频和二次调频

### 2.2 风力发电系统

风力发电是将风的动能转化为电能的技术。现代大型风电场的单机容量已达 10MW 以上。

#### 风力发电机组分类

- **陆上风电**：安装成本低（约 6000-8000 元/kW），维护方便
- **海上风电**：风资源更好（利用小时数 3000-4000h），但安装成本高（约 12000-18000 元/kW）
- **分散式风电**：靠近负荷中心，减少输电损耗

#### 风电并网关键技术

风电并网面临的主要挑战：

1. **出力预测**：短期预测精度需达到 85-90%，超短期预测精度需达到 90-95%
2. **电压控制**：风电场需配置动态无功补偿装置（如 SVG/STATCOM）
3. **次同步振荡（SSO）**：风电场经串联补偿线路并网时可能引发 SSO 问题
4. **电网适应性**：需满足电网频率偏差 ±2Hz 内正常运行的要求

### 2.3 储能系统

储能系统是解决新能源波动性和间歇性问题的关键技术。

#### 储能技术对比

| 技术类型 | 能量密度 (Wh/kg) | 功率密度 (W/kg) | 循环寿命 (次) | 效率 (%) | 应用场景 |
|----------|------------------|-----------------|--------------|----------|----------|
| 锂离子电池 | 150-280 | 300-2000 | 3000-8000 | 90-97 | 调频、调峰、用户侧 |
| 液流电池 | 25-50 | 50-200 | 10000-20000 | 70-85 | 大规模储能 |
| 压缩空气 | - | - | 20000+ | 40-55 | 削峰填谷 |
| 飞轮储能 | 5-50 | 500-5000 | 100000+ | 85-95 | 调频、电能质量 |
| 超级电容 | 5-10 | 5000-50000 | 1000000+ | 85-98 | 功率补偿 |

## 3. 电力系统优化与控制

### 3.1 经济调度

电力系统经济调度（Economic Dispatch, ED）是在满足负荷需求和运行约束的条件下，合理安排各发电机组的出力，使总发电成本最小。

数学模型：

```
min Σ Fi(Pi) = Σ (ai·Pi² + bi·Pi + ci)

s.t.
  Σ Pi = PLoad + PLoss          (功率平衡约束)
  Pi_min ≤ Pi ≤ Pi_max          (发电机出力约束)
  |Pi - Pi0| ≤ Ri·Δt            (爬坡速率约束)
  Pline_k ≤ Pline_k_max         (线路传输约束)
```

其中：
- `Fi(Pi)` 为第 i 台发电机的成本函数
- `ai, bi, ci` 为成本系数
- `PLoad` 为系统总负荷
- `PLoss` 为网损
- `Ri` 为爬坡速率

### 3.2 最优潮流

最优潮流（Optimal Power Flow, OPF）是在经济调度基础上增加网络约束，考虑线路传输功率限制和节点电压约束，求解最优运行方案。

OPF 的求解方法包括：
1. **线性规划法（LP）**：将非线性问题线性化，求解速度快
2. **内点法**：处理大规模非线性规划问题的有效方法
3. **遗传算法**：全局搜索能力强，适合非凸问题
4. **粒子群优化**：收敛速度快，参数少

### 3.3 需求响应

需求响应（Demand Response, DR）通过价格信号或激励机制引导用户调整用电行为，实现削峰填谷。

**需求响应分类：**

- **基于价格的需求响应（PDR）**：
  - 分时电价（TOU）
  - 实时电价（RTP）
  - 尖峰电价（CPP）

- **基于激励的需求响应（IDR）**：
  - 直接负荷控制（DLC）
  - 可中断负荷（IL）
  - 需求侧竞价（DSB）

**需求响应效果评估指标：**
- 削峰率 = (峰荷基线 - 响应后峰荷) / 峰荷基线 × 100%
- 节能率 = (基线用电量 - 响应后用电量) / 基线用电量 × 100%
- 用户满意度评分（1-5 分）

## 4. 电力市场

### 4.1 电力市场体系

现代电力市场通常采用"中长期+现货"的双轨制模式：

| 市场类型 | 交易周期 | 价格机制 | 功能 |
|----------|----------|----------|------|
| 中长期市场 | 年/月/周 | 协商定价/集中竞价 | 规避价格风险 |
| 日前市场 | 日前 | 边际出清 | 确定日前发电计划 |
| 实时市场 | 15min/5min | 实时出清 | 平衡实时偏差 |
| 辅助服务市场 | 月/日 | 补偿定价 | 保障电网安全 |

### 4.2 现货市场出清机制

电力现货市场采用节点边际电价（Locational Marginal Price, LMP）机制：

```
LMPi = λ + η·(∂PLoss/∂Pi) + Σ μk·(∂Pk/∂Pi)

其中：
  λ    — 系统边际能量价格
  η    — 网损灵敏度系数
  μk   — 线路 k 的阻塞影子价格
  ∂Pk/∂Pi — 节点 i 对线路 k 的功率传输分布因子（PTDF）
```

### 4.3 新能源参与电力市场

新能源参与电力市场的主要模式：

1. **保障性收购模式**：政府定价，电网全额收购
2. **绿色证书交易模式**：与绿证配合参与市场
3. **现货市场报量不报价模式**：以价格接受者参与
4. **报量报价模式**：完全参与市场竞争

## 5. 碳中和与能源转型

### 5.1 双碳目标

中国提出"2030 年碳达峰、2060 年碳中和"的双碳目标，电力行业是碳排放的主要来源之一，约占全国碳排放的 40%。

**电力行业减排路径：**
1. 能源结构转型：非化石能源占比 2030 年达 25%，2060 年达 80%+
2. 提升能效：煤电机组平均供电煤耗降至 300 gce/kWh 以下
3. 碳捕集利用与封存（CCUS）：2030 年示范应用，2050 年规模化
4. 电气化：终端电气化率 2030 年达 35%，2060 年达 70%

### 5.2 虚拟电厂

虚拟电厂（Virtual Power Plant, VPP）通过信息通信技术将分布式能源、储能、可调负荷等资源聚合起来，作为一个整体参与电网调度和电力市场。

VPP 运营模式：
- **负荷聚合**：整合可调节负荷（空调、热水器、充电桩）
- **分布式发电聚合**：整合分布式光伏、分散式风电
- **储能聚合**：整合用户侧和电网侧储能
- **混合模式**：同时聚合源-荷-储资源

### 5.3 综合能源系统

综合能源系统（Integrated Energy System, IES）通过多种能源形式的互补和梯级利用，提升整体能源利用效率。
